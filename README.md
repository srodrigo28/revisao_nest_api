#### Instalação Global
* 1. Verficar se tem o nodejs no computador
```
node --version
```
* 2. Instalar o NestJS como acesso global
```
npm i -g @nestjs/cli
```
* 3. Criando promeiro projeto
```
nest new introducao
```

* 4 Rodando o projeto
```
npm start
```
